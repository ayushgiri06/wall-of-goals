# 🌟 Wall of Goals

A motivational web app where anyone can write their personal goals and others can like and comment to show support — creating a virtual wall of inspiration! ✨

---

## 🚀 Features

- 📝 Post your goals
- 👍 Like goals shared by others
- 💬 Leave encouraging comments
- 🖥️ Built using pure **HTML**, **CSS**, and **JavaScript**

---

## 🎯 Why I Built This

I wanted to create something simple yet powerful — a place where people can share their dreams, big or small, and feel inspired by others. This project also helped me improve my DOM manipulation, form handling, and interactive UI design skills.

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
